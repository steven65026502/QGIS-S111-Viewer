# S-111 箭頭邊框寬度修正報告

## 修正日期
2024年12月19日

## 問題描述
箭頭符號顯示為全黑色，主要顏色被邊框覆蓋。

## 根本原因
邊框寬度設置不當，導致黑色邊框覆蓋了箭頭的主要顏色。

## S-111 規範依據

根據使用者提供的官方S-111規範信息：

### 邊框顏色要求
- **箭頭的邊框必須是黑色的** (符合S-111標準)

### 邊框寬度規範
- S-111 **附錄G (Scalable Vector Graphics (SVG) Coding)** 中的範例SVG程式碼
- 箭頭符號的筆劃寬度 (`stroke-width`) 設定為 **0.32**
- 單位：**毫米 (mm)**
- 這與其他尺寸定義一致（最大高度10mm，最大寬度4mm）

### 規範性質
- 附錄G內容為**資訊性 (Informative)**
- 數值為建議性，可根據實際使用調整
- 發生衝突時，**呈現目錄 (Portrayal Catalogue)** 優先

## 修正內容

### 1. 更新邊框寬度常數
```python
# 修正前
BORDER_WIDTH = 0.1  # 過小，可能不符合規範

# 修正後  
BORDER_WIDTH = 0.32  # 根據S-111附錄G SVG範例中的stroke-width建議值
```

### 2. 統一所有符號設置
- ✅ `apply_s111_standard_symbology()` 主要符號方法
- ✅ `apply_simple_arrow_symbology()` 備用符號方法  
- ✅ 最後備用方案的單一符號設置

### 3. 符號屬性設置
```python
symbol = QgsMarkerSymbol.createSimple({
    'name': 'arrow',
    'color': color,  # S-111規範顏色
    'outline_color': S111Standards.BORDER_COLOR,  # 黑色邊框
    'outline_width': str(S111Standards.BORDER_WIDTH),  # 0.32mm邊框寬度
    'outline_width_unit': 'MM',
    'size': '8',
    'size_unit': 'MM'
})
```

3. **最終修正**: 0.32mm - 符合S-111附錄G建議值
