## 🔧 技術實現細節

### 核心類別修改

#### 1. S111Standards 類增強
```python
class S111Standards:
    # 速度帶顏色方案 (根據 S-111 標準 Table 9-2 - 日間顯示顏色)
    # 使用官方規範的 RGB 顏色值
    SPEED_BANDS = [
        (0.0, 0.5, '#7652E2'),     # Band 1: 紫色 (RGB: 118, 82, 226)
        (0.5, 1.0, '#4898D3'),     # Band 2: 深藍色 (RGB: 72, 152, 211)
        # ... 其他速度帶
    ]
    
    # 黑色邊框顏色 (根據 S-111 標準要求)
    BORDER_COLOR = '#000000'  # 黑色邊框，確保在背景顏色相似時箭頭仍可見
    BORDER_WIDTH = 0.5  # 邊框寬度 (mm)
```

#### 2. 符號渲染系統重構
- **渲染器**: `QgsRuleBasedRenderer` (支援多速度帶)
- **符號類型**: SVG 符號 + 自定義箭頭混合方案
- **降級機制**: SVG 失敗時自動使用自定義箭頭
- **顏色控制**: 完全動態的顏色和邊框設定

#### 3. 備用方案完善
- 將原始的單色箭頭升級為多色規則系統
- 確保備用方案也完全符合 S-111 規範
- 實現相同的顏色和邊框要求
